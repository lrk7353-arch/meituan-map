# 大学城美食地图

一个面向高校学生的本地化美食发现平台，帮助解决"吃什么"的决策困境。

## 功能特性

### 🎯 核心功能
- 🎲 **三种决策模式** - 控制热量/控制预算/随意选择
- 🗺️ **地图探探模式** - 餐厅信息以气泡形式显示在地图实际位置
- 👆 **直观选择** - 底部滑动式选择（喜欢/不喜欢）
- 🎰 **幸运转盘** - 随机抽取美食品类，解决选择困难
- 🛵 **美团衔接** - 支持跳转到美团团购/外卖

### 📱 用户体验
- ⭐ 评价和评分系统
- 💾 收藏功能（本地存储持久化）
- 🎯 学生专属推荐
- 💰 价格档位分类
- 📍 基于位置推荐

## 技术栈

### 前端
- Vue.js 3 (Composition API)
- 高德地图 API
- Element Plus
- Pinia
- Vite

### 后端
- Node.js
- Express
- RESTful API

## 快速开始

### 后端启动
```bash
cd backend
npm install
npm run dev
```

### 前端启动
```bash
cd frontend
npm install
npm run dev
```

### 设计理念
1. **地图为核心** - 餐厅信息以气泡形式显示在地图实际位置
2. **感性决策** - 探探式滑动选择，符合真实用餐决策场景
3. **场景化推荐** - 根据热量/预算/心情智能推荐
4. **生态衔接** - 与美团团购/外卖无缝对接
