# 大学城美食地图 - 项目总结

## 项目概述

"大学城美食地图"是一个面向高校学生的本地化美食发现平台，旨在解决学生在大学城周边面临"吃什么"的决策困境。该项目集成了地图浏览、餐厅信息展示、搜索筛选、评价系统等核心功能。

## 技术栈

### 前端
- **Vue.js 3** - 使用 Composition API 构建
- **Vue Router** - 路由管理
- **Pinia** - 状态管理（支持本地存储持久化）
- **Element Plus** - UI组件库
- **Vite** - 构建工具
- **Axios** - HTTP 请求

### 后端
- **Node.js** - 运行时环境
- **Express** - Web框架
- **RESTful API** - API设计风格
- **JSON文件存储** - 轻量化数据存储

### 第三方服务
- **高德地图 API** - 地图展示和地理位置服务
- **Lorem Picsum** - 图片占位服务

## 部署说明

### 开发环境
```bash
# 后端
cd backend
npm install
npm run dev

# 前端
cd frontend
npm install
npm run dev
```

### 生产环境
```bash
# 前端构建
cd frontend
npm run build

# 后端部署（使用PM2）
cd backend
npm install -g pm2
pm2 start server.js --name meituan-backend
```
